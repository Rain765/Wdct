
  # Web Document Comparison Tool (Community)

  This is a code bundle for Web Document Comparison Tool (Community). The original project is available at https://www.figma.com/design/mg3gCnZGZVNFaqO9Skp4rI/Web-Document-Comparison-Tool--Community-.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the Next.js development server.

## Scripts

- `npm run dev` - start the development server
- `npm run build` - create a production build
- `npm run start` - run the production server
- `npm run lint` - run Next.js ESLint checks
